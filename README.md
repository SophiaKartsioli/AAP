AAP (stands for Artist Alley Portal)

This website built for astist who takes part in conventions and other events to diplay photographs and images from their booths, thus they build their own portfolio which will be publicly visible. Also the website can provide the latest news and having annoucments.

--Features--

-User authentication (login / signup) -Fast and responsive UI -CRUD functionality -API integration -Secure data handling

--GETTING STARTED--

This project is containerized using **Docker** and **Docker Compose**.

Prerequisites

Before running the application, make sure you have **one of the following installed**:

-**Docker Desktop** (recommended)  
  https://www.docker.com/products/docker-desktop/
-**Docker Engine + Docker Compose**

You can verify Docker is installed by running:

```bash
docker --version
docker-compose --version
```

Make sure you are in the root directory of the project, then run:

docker-compose up --build


Open your browser and visit:
http://localhost:8080
